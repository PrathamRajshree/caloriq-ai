# Caloriq – Food Image Classification Model

## Overview
This module contains the machine learning pipeline used in Caloriq
to classify Indian food items from images and estimate calories.

## Model
- Architecture: CNN (TensorFlow / Keras)
- Input size: 224×224 RGB
- Classes: Indian food categories
- Output: Food label + confidence score
